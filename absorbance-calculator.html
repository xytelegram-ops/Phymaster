📖 使用方法

1. 创建样品并上传文件

  • 点击「+ 新建」创建样品（如"样品 A"、"样品 B"）
  • 在下拉框中选择当前样品，然后上传该样品的 txt 文件（可多选）
  • 每个文件需包含两列：波长 吸光度（制表符/空格/逗号分隔）
  • 切换到另一个样品，继续上传——各样品文件互不混淆

2. 设置反应时间

  • 在每个文件的输入框中填写反应时间（数值）
  • 点击预设按钮快速填入（每个文件的预设值不同，按序号递增）
  • 时间最小的文件自动成为该样品的基准（通常 t = 0）

3. 设置参数

  • 波长范围：选择用于计算吸光度最大值的波长区间
  • 点击「自动填充」可快速填入所有数据覆盖的波长范围
  • 常数：性能 = Δ吸光度 / 常数，根据实验需要输入
  • 单位：可选填时间单位和性能单位，会显示在图表和表格中

4. 查看结果

  • 点击「开始计算」或修改任何参数后自动更新（500ms 防抖）
  • 性能汇总图：所有样品的性能曲线叠加在一张图上，支持切换折线图/柱状图/散点图
  • 支持切换折线图 / 柱状图 / 散点图
  • 各样品光谱图：每个样品独立显示原始光谱，蓝色区域标注波长范围
  • 数据表：按样品分页，列出每个时间点的最大吸光度和性能值

5. 导出

  • 每个图表可导出为 PNG 图片
  • 取消勾选「网格」后再导出可获得无网格线的干净图片
  • 「导出 CSV」下载完整数据表格

计算原理

  • 对每个样品，以时间最小的文件为基准，计算各时间点的 Δ吸光度
  • 性能 = Δ吸光度 / 常数统一换算
  • 基准文件的吸光度在指定波长范围内取最大值，其他文件在相同波长处取值
💡 提示：修改任何参数后图表会自动更新（500ms 防抖），无需手动点击按钮。X/Y 轴标签支持手动输入，默认使用英文以避免图表中文显示问题。时间单位可在参数设置中填写，自动显示在图表坐标轴上。
