📖 使用方法

1. 数据输入

  • 手动输入：在文本框中每行输入 波长(nm) 数值（空格或 Tab 分隔）
  • 文件上传：支持 .txt / .csv / .xlsx 格式，每行两列数据
  • 示例数据：内置 ZnO 薄膜典型透光光谱（直接带隙约 3.25 eV）

2. 参数设置

  • 数据类型：选择「透光率 T%」「吸光度 A」或「吸收系数 α (cm⁻¹)」，程序自动转换
  • 带隙类型：直接带隙 (n=1/2)、间接带隙 (n=2)、直接禁戒 (n=3/2)
  • 材料预设：可选择 ZnO、TiO₂ 等常见材料快速填充参数
  • 薄膜厚度：用于吸收系数与吸光度之间的换算
  • 拟合范围：设置 Tauc plot 线性拟合的能量区间

3. 查看结果

  • 点击「计算带隙」或修改参数后自动触发计算
  • Tauc 曲线图：(αhν)^(1/n) vs hν，带线性拟合线和 x 轴截距标注
  • 带隙能量：显示拟合得到的带隙值 (eV) 和拟合优度 R²
  • 支持导出拟合数据为 CSV 文件

Tauc Plot 原理

  • 公式：(αhν)^(1/n) = A (hν − E_g)
  • 透光率 → 吸光度：A = −log₁₀(T/100)
  • 吸收系数：α = 2.303 × A / d（d 为膜厚）
  • 在 Tauc plot 的线性区域做切线，外推至 x 轴截距即为带隙 E_g
💡 提示：默认显示 ZnO 示例数据（直接带隙）。如需测量自己的样品，请确保数据类型与输入数据匹配（透光率 T%、吸光度 A 或吸收系数 α）。拟合范围的选择对结果影响较大，建议根据 Tauc 曲线的线性区域手动调整。
